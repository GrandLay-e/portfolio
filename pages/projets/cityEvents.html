CITYEVENTS

CityEvents

PHP Symfony MySQL HTML/CSS JavaScript

🤝 Projet collaboratif - Application web full-stack développée en équipe avec le framework Symfony pour la gestion et la découverte d'événements urbains. Plateforme interactive permettant aux utilisateurs de créer, consulter et participer à des événements dans leur ville.

👥 Travail d'équipe et organisation :

  • Développement collaboratif Coordination avec GitHub (branches, pull requests, code reviews)
  • Communication Documentation partagée, standards de code communs

🔧 Fonctionnalités principales :

  • Gestion d'événements Création, modification et suppression d'événements par les utilisateurs
  • Authentification utilisateur Système complet d'inscription, connexion et gestion de profils
  • Recherche et filtrage Recherche par date, catégorie, localisation et mots-clés
  • Participation aux événements Inscription et suivi des événements
  • Système de catégories Classification des événements (concerts, sports, culture, etc.)
  • Interface responsive Design adaptatif pour mobile et desktop
  • Base de données relationnelle Modélisation avec Doctrine ORM et MySQL
  • Gestion des médias Upload et affichage d'images pour les événements
  • Notifications Système d'alertes pour les événements à venir
  • Panel administrateur Interface d'administration pour la gestion du contenu
