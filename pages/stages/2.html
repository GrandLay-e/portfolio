Sécurisation des uploads et renommage fichiers

J'ai sécurisé l'accès aux fichiers uploadés puis implémenté un renommage horodaté pour éviter les écrasements entre documents homonymes.

function cleanStr($filename) {
    $filename = iconv('UTF-8', 'ASCII//TRANSLIT//IGNORE', $filename);
    $filename = preg_replace('/[^a-z0-9]/', '-', $filename);
    return trim(preg_replace('/-+/', '-', $filename), '-');
}
