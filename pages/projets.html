MES PROJETS

Company Scraper

Python

Ce projet est un scraper d'entreprises qui utilise Python et sélénium pour extraire des informations sur les entreprises à partir d'une source web. Il permet de collecter des données telles que le nom, l'adresse et d'autres détails afin de les stocker ou de les analyser. Il y a aussi une possibilité de postuler à des offres d'emploi en utilisant les données collectées.

Eyrein Product

Python

Ce projet esr réalisé dans le cadre de mon stage de première année de BTS SIO. Il s'agit d'un outil pour télécharger les fiches produits d'une entreprise, Eyrein, à partir de leur site web. Les fichiers sont, par la suite, stockés et classés dans des dossiers organisés par produits incluant un fichier ZIP qui comprend les différents fiches d'un produit.

Pokedex

PHP HTML/CSS SQL

Ce projet présente un Pokedex utilisant HTML, PHP et SQL. Il permet d'ajouter, de supprimer, de modifier et d'afficher des pokemons depuis une API externe (Tyradex) et de les stocker dans une base de données SQL.

FileArranger

Python

Outil d'organisation automatique de fichiers d'un dossier que vous lui passez. L'organisation se fait par extension dans un premier temps puis par Années et par mois.

Jeu du Pendu

Python

Le projet "Jeu du Pendu" est une application console qui récupère des mots aléatoires à partir d'une source web pour permettre à l'utilisateur de jouer au jeu du pendu. Dans ce jeu, un mot est choisi aléatoirement, et l'utilisateur doit deviner ce mot en proposant des lettres.

Plus ou Moins

C Sharp

Le projet "PlusOuMoins" est un jeu de devinette en console où le joueur doit deviner un nombre aléatoire généré par l'ordinateur. Le programme utilise la classe Random pour sélectionner un nombre entre 1 et 100.
